EBProjectFramework Project:
This is framewrok Project. It has basrSCripts, config, driver, Helpers, Logs and Screenshot folder. It's capabilities includes Browser operations, File IO oprations, writing Logs and taking screenshots.

For Adding New Test(EverBridge_Demo Project)
1) Create a new Test file under //TestCases folder or can add Test Methods in the existing Test file.
2) All WebElements are under //pageObjects
3) If any new Property needs to pass to the scripts, use //configFiles folder.
4) Input values are coming from a Input file(Path //testData)


NOTE:
1) Logs can be found under EBProjectFramework//Logs folder
2) Screenshot for failed test cases can be Found EBProjectFramework//Screenshots
